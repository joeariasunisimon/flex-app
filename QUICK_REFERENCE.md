## 💾 Database Operations

### Create Game
```kotlin
val createGameUseCase = CreateGameUseCase(gameRepository)
val game = createGameUseCase("Game Name", "5 in a row")
```

### Add Bingo Card
```kotlin
val card = BingoCard(gameId = 1, numbers = listOf(1,2,3,...90))
bingoCardRepository.insertCard(card)
```

### Mark Number
```kotlin
val marked = MarkedNumber(gameId = 1, number = 42)
markedNumberRepository.insertMarkedNumber(marked)
```

### Get Game Data
```kotlin
val game = gameRepository.getGameById(1)
val cards = bingoCardRepository.getCardsByGameId(1)
val marked = markedNumberRepository.getMarkedNumbersByGameId(1)
```

## 🎮 Game Rules Setup

The app supports three win conditions:
1. **5 in a row** - Traditional bingo (horizontal, vertical, or diagonal line)
2. **X** - Cross/diagonal pattern
3. **Full card** - All 90 numbers marked

## 🐛 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Build fails with SQLDelight error | Run `generateCommonMainBingoDatabaseInterface` |
| Navigation not working | Check Route.kt for typos |
| ViewModel state not updating | Verify StateFlow collection in UI |
| Database empty | Check database driver initialization |
| UI not rendering | Verify Composable signature and parameters |
