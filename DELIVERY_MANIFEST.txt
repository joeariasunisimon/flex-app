================================================================================
                         BUILD INSTRUCTIONS
================================================================================

Prerequisites:
- Android Studio / IntelliJ IDEA
- Kotlin 2.3.20+
- Java 11+
- Android SDK 36

Building:
1. Open project in Android Studio
2. Gradle will sync automatically
3. Run: ./gradlew build (full build)
4. Or: ./gradlew :composeApp:assembleDebug (Android APK)

Running on Device:
1. Connect Android device with USB debugging
2. Run: ./gradlew :composeApp:installDebug
3. Or use "Run" in Android Studio
